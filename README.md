# Slovenian Tarok

A browser-based implementation of the Slovenian 4-player Tarok card game.

## Requirements

- [Node.js](https://nodejs.org/) (v18 or later)

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Build (optional)

```bash
npm run build
npm run preview
```
